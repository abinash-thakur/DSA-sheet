# Combinatorics

Combinatorics problems can usually be solved using one of these three approaches:

1. Dynamic Programming (DP)
2. Pure Mathematics
3. Hybrid (DP + Math)

## Core Math Concepts to Master

1. Factorial (`n!`)
2. `nCr` and `nPr`
3. Inverse Factorial
4. Modular Arithmetic

## Quick Notes

- Use DP when states and transitions are clear.
- Use math formulas when direct counting is possible.
- Use hybrid methods when constraints require both counting logic and optimization.
